# Description
This project follows the starter guide Spotify Display, but it has one feature that makes it unique, which is a rotary encoder that is being used. This rotary encoder is used for the volume control.

# Motivation
After looking at all the starter projects, the spotify display seemed to be very interesting and also useful, which is why I decided myself for this starter project.
