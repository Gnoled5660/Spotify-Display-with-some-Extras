# Description
This project follows the starter guide Spotify Display, but it has two feature that makes it unique, which is a rotary encoder that is being used and a small 0.96 inch display. This rotary encoder is used for the volume control and the display is being used to show the current time.

# Motivation
After looking at all the starter projects, the spotify display seemed to be very interesting and also useful, which is why I decided myself for this starter project. I also found that it had a lot of potential to expand this idea and use it to create something unique.
